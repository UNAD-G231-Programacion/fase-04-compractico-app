# `/app` — Espacio de Trabajo Colaborativo

Aquí los colaboradores depositan todos los archivos y carpetas generados durante el desarrollo del proyecto. Estos deben usar nombres descriptivos, en minúsculas y sin espacios, se puede usar guiones bajos para separar (ej: `mi_archivo.py`).

---

*Para información general del proyecto, consulta el `GuiaColaboracion.md` en la raíz del repositorio.*
